<task type="auto">
  <name>Task 2: Create Button and Card UI components</name>
  <files>src/components/ui/Button.astro, src/components/ui/Card.astro</files>
  <action>
    **Button.astro** (src/components/ui/Button.astro):
    - Props interface: `variant` ('primary' | 'secondary', default 'primary'), `size` ('sm' | 'md' | 'lg', default 'md'), `href` (optional string — renders as `<a>` if provided, `<button>` otherwise)
    - Base styles:
      - display inline-block, border --border-width solid --color-text
      - background --color-bg, color --color-text
      - font-family --font-body, font-weight 700, text-transform uppercase, letter-spacing 0.05em
      - cursor pointer, text-decoration none (for anchor variant)
      - box-shadow: --shadow-offset-md --shadow-offset-md 0 --color-accent (primary) or --color-accent-warm (secondary)
      - transition: transform 0.15s ease, box-shadow 0.15s ease
    - Hover/focus-visible: translate(-2px, -2px), shadow increases to --shadow-offset-lg
    - Active: translate(0, 0), shadow decreases to --shadow-offset-sm
    - Size variants: sm (space-xs/space-md padding, font-sm), md (space-sm/space-lg, font-md), lg (space-md/space-xl, font-lg)
    - Use `<slot />` for button content

    **Card.astro** (src/components/ui/Card.astro):
    - Props interface: `accent` ('primary' | 'secondary' | 'amber', default 'primary') — controls shadow color
    - `<article class="card card-{accent}">` wrapping `<slot />`

      - border --border-width solid --color-border
      - background --color-bg-elevated
      - box-shadow: --shadow-offset-md --shadow-offset-md 0 with color based on accent prop
      - Hover: translate(-2px, -2px), shadow offset increases to --shadow-offset-lg
      - transition: transform 0.15s ease, box-shadow 0.15s ease
    - Accent colors: primary -> --color-accent, secondary -> --color-accent-warm, amber -> --color-accent-amber
