<task type="auto">
  <name>Task 2: Create design token system and global styles</name>
  <files>src/styles/tokens.css, src/styles/global.css</files>
  <action>
    Create `src/styles/tokens.css` with CSS custom properties on `:root`:

    **Colors:**
    - `--color-bg`: #0a0a0a (near-black background)
    - `--color-bg-elevated`: #141414 (slightly lighter for cards/surfaces)
    - `--color-text`: #ffffff (white text)
    - `--color-text-muted`: #a0a0a0 (subdued text)
    - `--color-accent`: #00ff41 (neon green — primary accent)
    - `--color-accent-warm`: #ff6b35 (orange — secondary accent)
    - `--color-accent-amber`: #ffbf00 (amber — tertiary accent)
    - `--color-border`: #ffffff (white borders for neo-brutalist look)

    **Typography:**
    - `--font-heading`: 'Space Grotesk', sans-serif
    - `--font-body`: 'Space Mono', monospace
    - Font size scale using major third (1.25 ratio): --font-xs through --font-3xl
    - Base font size: 16px (1rem)

    **Spacing (8px grid):**
    - `--space-xs`: 0.5rem (8px) through `--space-2xl`: 4rem (64px)

    **Neo-brutalist tokens:**
    - `--border-width`: 3px
    - `--shadow-offset-sm`: 2px, `--shadow-offset-md`: 4px, `--shadow-offset-lg`: 6px
    - `--radius`: 0 (no border radius — brutalist)

    Create `src/styles/global.css` with:
    - Modern CSS reset (box-sizing border-box on all elements, margin/padding reset, img max-width)
    - Body: dark background (--color-bg), white text (--color-text), font-family (--font-body)
    - Headings: font-family (--font-heading), font-weight 900
    - Links: color inherit, no underline by default
    - Selection: accent-colored background
    - Smooth scrolling on html
    - Import Bunny Fonts is NOT done here — it goes in the layout head tag (Plan 02). Tokens and global.css are pure CSS with no external dependencies.
  </action>
  <verify>Files exist at src/styles/tokens.css and src/styles/global.css. Tokens file contains --color-accent, --font-heading, --space-md, --border-width, --shadow-offset-md. Global file contains box-sizing, body styling, heading styling.</verify>
  <done>Complete design token system with colors, typography, spacing, and neo-brutalist visual tokens. Global CSS reset with dark mode base styles. No hard-coded values — everything references tokens.</done>
